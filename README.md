# Homebrew-gnuradio

## Introduction
This tap provides a few extra blocks for gnuradio. As of now, gnuradio is installed
via default repository, together with librtlsdr.

* gr-baz
* gr-dsd
* gr-fosphor
* gr-op25
* gr-osmosdr
* gr-tpms (WIP)

In addition:

* gqrx
* dump1090

Most formulas are HEAD-only.
